# DistrictIntel

> AI-powered research platform for K–12 school district facilities and construction intelligence.

## Vision
DistrictIntel is a long-term software project focused on collecting, researching, validating, and organizing information about every public school district in Ohio.

The platform is intended to help identify future construction opportunities by researching facility age, renovations, bond history, capital planning, enrollment trends, and other indicators.

## Guiding Principles
- Every fact must have a source.
- AI is an analyst, not a search engine.
- Prefer primary sources.
- Never invent information.
- Build for maintainability over speed.
- Keep modules focused on a single responsibility.

## Technology
- Python 3.12+
- SQLite
- OpenAI API
- VS Code
- GitHub
- GitHub Codespaces

## Long-Term Goal
A searchable intelligence platform that can answer questions about Ohio school districts and identify future construction opportunities.
